> 由 **玄宇绘世设计工作室** 荣誉出品。

> 一款专业级系统磁盘深度清理工具，专为解决C盘空间不足与电脑冗余文件堆积设计。纯本地运行，无弹窗、无后台，致力于提供最干净利落的系统维护体验。

## ✨ 核心特色功能
- ⚡ **C 盘一键静默瘦身**：一键扫描并清理系统缓存、更新补丁、回收站，带有安全的默认勾选保护防呆机制。
- 💬 **专项缓存深度识别**：内置特殊规则路径，精准识别清理 微信、QQ、企微、钉钉、爱奇艺、优酷、B站 等本地难缠缓存。
- 📊 **动态阈值大文件检索**：支持跨越所有盘符检索，用户可自定义容量阈值（如扫描 >1GB 的文件），瞬间找出遗忘的系统庞然大物。
- 🔄 **MD5 重复文件精准查重**：采用双层过滤算法（大小初筛 + MD5特征码校验）。自带机械硬盘保护提示机制，拒绝伤盘的无效 I/O 运算。
- 🛡️ **工程级交互体验**：所有操作均需显式勾选确认；支持右键预览文件目录；支持对文件大小/时间戳的双重防爆栈安全排序。

## 💻 本地运行与二次开发
1. 确保安装了 Python 3.8 或以上版本。
2. 安装界面依赖库：`pip install PySide6`
3. 运行主程序：`python main.py`
*(注：为保证能顺利扫描受保护的 Windows 系统目录，建议以管理员身份运行开发环境终端)*

### 🛠️ 编译为独立 EXE
在代码根目录下放置 `icon.ico` 后，运行以下命令即可打包为包含图标的无控制台版本：
```bash
pyinstaller -F -w -i icon.ico --add-data "icon.ico;." --add-data "logo.png;." --name XYD-磁盘管家PRO main.py
```
